Descripción del Proyecto

Este proyecto de práctica tiene como objetivo desarrollar una aplicación web que permita a los usuarios gestionar sus tareas diarias de manera eficiente. La aplicación contará con funcionalidades como la creación, edición y eliminación de tareas, así como la posibilidad de organizarlas por categorías y establecer recordatorios.
